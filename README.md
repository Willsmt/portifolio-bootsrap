# 🚀 Portfólio Fullstack - Willians Martins

Este é o código-fonte do meu portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades técnicas e minha trajetória como desenvolvedor Fullstack. O site foi desenhado com foco em uma identidade visual moderna (utilizando as fontes **JetBrains Mono** e **Syne**) e responsividade com **Bootstrap**.

## 💻 Sobre o Projeto

O projeto é uma página web de apresentação pessoal (One Page) que reflete meu dia a dia como desenvolvedor, destacando:
* **Back-end robusto:** Focado em Python, FastAPI e Django.
* **Automações:** Experiência prática com n8n para otimização de processos.
* **UI/UX:** Design e prototipação feitos no Figma.
* **Identidade Visual:** Uma estética "code-centric" com elementos interativos e uma representação visual de código em Python na seção *Hero*.

## 🛠 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **Front-end:** HTML5, CSS3, JavaScript.
* **Framework CSS:** Bootstrap 5.3.3.
* **Fontes:** Google Fonts (JetBrains Mono, Syne).
* **Ícones:** Bootstrap Icons.
* **Versionamento:** Git & GitHub.

## 📁 Estrutura do Repositório

* `index.html`: Estrutura principal do site e marcação semântica.
* `main.css`: Estilização personalizada, animações de entrada e layout responsivo.
* `script.js`: Lógica para interações e comportamento da página.

## 🚀 Como Executar

Para visualizar o projeto localmente:

1.  Clone este repositório:
    ```bash
    git clone https://github.com/seuusuario/seu-repositorio.git
    ```
2.  Abra o arquivo `index.html` diretamente no seu navegador ou utilize a extensão **Live Server** do VS Code para uma melhor experiência de desenvolvimento.

## 🎯 Objetivo

O objetivo deste portfólio é documentar meu aprendizado contínuo, exibir os projetos que desenvolvi e facilitar o contato para oportunidades de trabalho, colaborações e desafios profissionais.
